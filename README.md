# Nirmal's Portfolio

A personal portfolio website for **Nirmal** (aka Devil) — a BCA student at Maa Pateshwari University, Balrampur, passionate about web development, programming, and exploring AI tools and new technologies.

The site is built with a dark, sketchbook-inspired theme — warm ink-black background, a single burnt-sienna accent color, and a mix of serif (Fraunces), monospace (JetBrains Mono), and sans-serif (Inter) typography.

## What's on the site

- **Hero** — quick intro: BCA Student · Web Developer · Programmer · Gamer
- **About** — background on Nirmal's interests in web dev, programming, gaming, graphic design, and AI
- **Skills** — grouped into Languages, CS Fundamentals, and Web & Tools
- **Projects** — a showcase of three builds:
  - *Personal Portfolio Website* — this site itself, fully responsive
  - *ResumeCraft* — a simple, ATS-friendly resume builder
  - *File Management System* — a Python CLI tool for managing text files
- **Education** — BCA coursework at Maa Pateshwari University, Balrampur
- **Contact** — a working contact form (emails land directly in Nirmal's inbox) plus links to email, GitHub, and Instagram
- **Back to Top** — a fixed button that appears while scrolling for quick navigation

## Tech stack

- **React + Vite** for the frontend
- **Vercel serverless functions** powering the contact form
- **Resend** for sending contact form submissions by email
- Custom CSS with a token-based design system for consistent theming
